Issues with FAST Linked Data API

  • Author: Amaresh Joshi
  • Date: September 14, 2021
  • The documentation is very poor. It contains many missing pieces and errors.
  • There is no reference code samples (like there is for the assignFAST API).
  • I sent in a request for code samples using their feedback form a few weeks ago and never got any reply.
  • The API page itself says the API is still experimental so I would hesitate to use it in production.
  • The biggest problem is that the results returned by the search call are not the most popular. They seem really obscure.
